Приложение: "Трекер воды" (Water Tracker)
Краткое описание
Мобильное приложение для отслеживания ежедневного потребления воды. Пользователь может устанавливать дневную норму, добавлять выпитые порции воды и отслеживать прогресс в виде прогресс-бара. Простой и понятный интерфейс с визуальной обратной связью.

                                      Основные функции (MVP)
Установка дневной нормы воды (в миллилитрах).
Добавление порции воды (стандартные или пользовательские объемы).
Отображение прогресса за текущий день (прогресс-бар, процент выполнения, выпитый объем).
История за день в виде списка добавленных порций.
Сброс прогресса в начале нового дня.
Получение мотивирующих цитат о здоровье из публичного API.

                                    Сценарии использования
Установка нормы:
Пользователь заходит в настройки → вводит норму → сохраняет.
Добавление порции:
На главном экране пользователь нажимает кнопку "250 мл" → прогресс-бар обновляется.
Либо нажимает "Добавить свою порцию" → вводит объем → добавляет.
Просмотр истории:
Пользователь переходит на экран истории → видит список всех порций за сегодня.
Мотивация:
При запуске приложения загружается случайная цитата о здоровье.
