# 💧 Water Tracker - Трекер воды

## 🎯 Основные функции

### 📊 **Отслеживание прогресса**
- Установка персонализированной дневной нормы
- Визуальный прогресс-бар с цветовой индикацией
- Процент выполнения и объем выпитой воды
- Автоматический сброс в начале нового дня

### 💧 **Удобное добавление воды**
- Быстрые кнопки стандартных объемов (250мл, 500мл, 750мл)
- Возможность ввода произвольного объема
- История всех приемов за день
- Временные метки каждой порции

### 🌟 **Дополнительные возможности**
- Мотивирующие цитаты о здоровье (обновляются ежедневно)
- Интеграция с погодным API для рекомендаций
- Локальное хранение данных
- Персонализация имени пользователя
- Рекомендации на основе погодных условий

## 🚀 Сценарии использования

### 🎯 **Установка дневной нормы**
1. Откройте **Настройки** (иконка ⚙️)
2. В разделе "Дневная цель" введите нужный объем (например, 2500)
3. Нажмите кнопку сохранения 💾
4. Вернитесь на главный экран - цель обновится

### 💧 **Добавление порции воды**
**Вариант 1 (Быстрый):**
1. На главном экране нажмите кнопку с нужным объемом (250мл, 500мл, 750мл)
2. Прогресс-бар автоматически обновится

**Вариант 2 (Произвольный объем):**
1. Нажмите кнопку "➕ Добавить воду"
2. Введите нужный объем в миллилитрах
3. Или выберите из шаблонов
4. Нажмите "ДОБАВИТЬ ВОДУ"

### 📊 **Отслеживание прогресса**
- **Прогресс-бар** меняет цвет: 🔴 красный (<30%), 🟡 желтый (30-70%), 🟢 зеленый (>70%)
- **Процент выполнения** отображается в центре круга
- **Текущий объем** и цель показываются под прогрессом
- **Мотивационные сообщения** меняются в зависимости от прогресса

### 📋 **Просмотр истории**
1. Нажмите кнопку "📋 История потребления"
2. Просмотрите статистику за сегодня
3. Увидите список всех приемов воды с временными метками
4. Проверьте общий объем за день

### 🌟 **Ежедневная мотивация**
- При запуске приложения загружается **новая цитата** о здоровье
- Цитаты обновляются **раз в 24 часа**
- В зависимости от погоды появляются **персонализированные рекомендации**
- **Совет дня** помогает выработать полезные привычки
